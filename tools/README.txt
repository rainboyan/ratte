1 需要安装Java 6
2 安装好后，在命令行运行
> run org.scribe.examples.DoubanExample

或者
> run org.scribe.examples.SinaExample

分别获取访问的Token和secret

其他：
163
> run org.scribe.examples.T163Example

Sohu:
> run org.scribe.examples.SohuExample

QQ:
> run org.scribe.examples.QqExample

Fanfou:
> run org.scribe.examples.FanfouExample

默认的API key是Ratte应用提供，你可以修改为自己的。
